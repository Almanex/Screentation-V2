# Руководство пользователя Screentation V2 — Как делать разметку скриншотов в Windows

> [!NOTE]
> Screentation V2 — это профессиональная настольная утилита, автоматизирующая процесс разметки, добавления аннотаций и нумерации шагов на снимках экрана. Приложение работает в фоне, перехватывая скопированные в буфер обмена скриншоты, и позволяет добавлять векторные фигуры, размывать данные, кадрировать или вырезать фрагменты. Экспортируйте файлы в высоком качестве WebP, PNG или JPEG без риска перезаписи существующих снимков.

Screentation V2 предоставляет удобное нативное окружение для технических писателей, разработчиков программного обеспечения и специалистов службы поддержки, создающих пошаговые инструкции. Благодаря работе в системном трее, программа существенно ускоряет процесс захвата, разметки и экспорта изображений.

## Основные возможности

### Мониторинг буфера обмена и системный трей
Screentation V2 не делает снимки экрана самостоятельно. Вместо этого оно отслеживает системный буфер обмена. Когда вы делаете скриншот стандартными средствами Windows (например, с помощью `Win + Shift + S` или `PrintScreen`), Screentation мгновенно импортирует его. Закрытие главного окна сворачивает приложение в трей, позволяя фоновому захвату продолжать работу.

### Премиальные инструменты векторной разметки
*   **Стрелка**: Добавление аккуратных указательных стрелок для выделения элементов интерфейса.
*   **Рамка (Прямоугольник)**: Очерчивание ключевых компонентов рамочками с возможностью включения полупрозрачной заливки.
*   **Текст**: Ввод произвольного текста поверх скриншота с возможностью регулировки размера шрифта и цвета.
*   **Маркер (Выделение)**: Полупрозрачная кисть для выделения строк кода, важных блоков текста или элементов дизайна.
*   **Динамические слайдеры цвета и толщины**: Изменение толщины пера и цвета в реальном времени; изменения мгновенно применяются к выделенной фигуре.

### Маскирование и скрытие данных
*   **Размытие по Гауссу**: Наложение эффекта размытия для скрытия паролей, персональных данных или конфиденциальной информации.
*   **Клонирующий штамп (Eraser)**: Маскирование нежелательных элементов путем копирования текстуры фона из одной области скриншота в другую.

### Умная обрезка и вырезание фрагментов
*   **Умная обрезка**: Кадрирование изображения под любой прямоугольный размер. Ранее созданные элементы разметки автоматически смещаются на нужный вектор, сохраняя свои позиции относительно обрезанного кадра.
*   **Вырезка фрагментов (Slice Cut)**: Удаление горизонтальной или вертикальной полосы из скриншота. Оставшиеся части изображения бесшовно соединяются, а все векторные элементы ниже или правее места выреза сдвигаются автоматически.

### Автоматическая нумерация шагов (Инструкции)
Размещение маркеров-кружков, которые автоматически нумеруются по порядку для создания пошаговых гайдов.
*   Поддерживаются три формата: цифры (`1, 2, 3...`), заглавные латинские буквы (`A, B, C...`) и строчные латинские буквы (`a, b, c...`).
*   Размер кружка с шагом динамически масштабируется при изменении толщины пера.
*   Начальное или текущее значение следующего шага можно изменить в любой момент через панель настроек.

### Форматы экспорта (WebP, PNG, JPEG)
Сохраняйте размеченные скриншоты в форматах **PNG**, **JPEG** или сжатом **WebP**.
*   Вы можете настроить любую удобную директорию сохранения.
*   Пакетный экспорт («Сохранить все») экспортирует всю ленту скриншотов одним кликом.
*   Имена файлов генерируются на основе реального индекса в списке (например, `Screentation_02.png`), что гарантирует защиту от случайной перезаписи существующих файлов.

## Быстрый старт

1.  **Запустите приложение**: Откройте Screentation V2. Вы увидите чистое рабочее пространство. Иконка программы также появится в системном трее.
2.  **Сделайте скриншот**: Используйте стандартную комбинацию клавиш Windows (например, `Win + Shift + S`) для захвата части экрана.
3.  **Добавьте разметку**: Screentation автоматически загрузит снимок из буфера. Выберите инструмент на левой панели (например, Стрелку, Маркер или Шаг) и нарисуйте его на холсте.
4.  **Сохраните результат**: Нажмите **Сохранить текущий** (или `Ctrl + S`) для экспорта текущего кадра, либо **Сохранить все** для экспорта всей сессии скриншотов в выбранную папку.

## Горячие клавиши

| Сочетание клавиш | Действие |
| --- | --- |
| `Ctrl + V` | Вставить скриншот из буфера обмена вручную |
| `Ctrl + Z` | Отменить последнее действие (Undo) |
| `Ctrl + Y` / `Ctrl + Shift + Z` | Повторить отмененное действие (Redo) |
| `Ctrl + S` | Сохранить активный скриншот |
| `Ctrl + Shift + S` | Сохранить все скриншоты (пакетное сохранение) |
| `Delete` / `Backspace` | Удалить выделенный элемент разметки |
| `Escape` | Сбросить активное выделение или выйти из режима рисования |
| `Enter` | Применить обрезку или вырезку фрагмента |
| `Escape` (в режиме обрезки) | Отменить обрезку или вырезку фрагмента |
| `1` | Инструмент «Рамка» |
| `2` | Инструмент «Шаг» |
| `3` | Инструмент «Стрелка» |
| `4` | Инструмент «Размытие» |
| `5` | Инструмент «Штамп» |
| `Ctrl + Колесо мыши` | Масштабирование холста относительно курсора |
| `Зажатие колесика мыши (перетаскивание)` | Перемещение по увеличенному изображению |

## Часто задаваемые вопросы (FAQ)

### Предупреждение Windows Defender SmartScreen при запуске?
Поскольку автономный установщик не подписан дорогостоящим сертификатом разработчика, Windows может выдать предупреждение при первом запуске. Нажмите ссылку **«Подробнее»**, затем кнопку **«Выполнить в любом случае»**, чтобы продолжить установку.

### Запуск нескольких копий приложения?
Screentation V2 является приложением одного экземпляра (Single-Instance). Повторный запуск программы перехватывает управление и переводит уже запущенную фоновую копию в трее на передний план, предотвращая дублирование иконок.

### Почему элементы управления иногда становятся белыми на белом?
В Screentation V2 встроен обработчик динамической смены системных тем. Если Windows переключается между темным и светлым режимом оформления во время работы программы, интерфейс мгновенно перерисует все кнопки и шрифты для сохранения полной читаемости.
