# Screentation

*Профессиональное десктопное приложение для захвата, разметки и нумерации шагов скриншотов на Windows.*

Screentation — это нативное Windows-приложение для редактирования и аннотирования скриншотов. **Обратите внимание:** Screentation не делает снимки экрана самостоятельно. Вместо этого вы создаете скриншот стандартными встроенными средствами Windows (например, с помощью `PrintScreen`, `Win+Shift+S` и т. д.), а Screentation автоматически перехватывает изображение из буфера обмена для мгновенной разметки.

Приложение написано на C# с использованием WinUI 3 (Windows App SDK) и графической библиотеки Win2D для аппаратного ускорения отрисовки. Оно позволяет мгновенно импортировать скриншоты из буфера обмена (даже когда приложение свернуто в системный трей), наносить поверх них стрелки, рамки, маркеры подсветки, текст, размывать конфиденциальные данные, обрезать изображения, вырезать фрагменты и автоматически нумеровать шаги инструкций (как цифрами, так и латинскими буквами).

Интерфейс пользователя полностью локализован на **три языка**: русский, английский и немецкий.

Для получения подробной информации об использовании приложения ознакомьтесь с [Руководством пользователя](GUIDE_RU.md).

## Основные возможности

* **Автоматический захват**: Фоновый мониторинг буфера обмена (даже в свернутом виде) автоматически обнаруживает и импортирует скриншоты, скопированные стандартными утилитами снимков экрана Windows.
* **Инструменты разметки**: Полупрозрачный маркер подсветки, рамки (с заливкой и без), указательные стрелки, размытие конфиденциальных данных по Гауссу, клонирующий штамп (Eraser) и текстовые блоки.
* **Автонумерация шагов**: Маркеры с автоинкрементом поддерживают числовой и буквенный форматы.
* **Умная обрезка**: Кадрирование изображения с автоматическим сдвигом и пересчетом координат ранее нарисованных элементов.
* **Локализация**: Полный перевод интерфейса на 3 языка (русский, английский, немецкий) с автоматическим определением языка системы и возможностью принудительного переключения через командную строку.

## Начало работы

### Требования

* ОС: Windows 10 (версия 1809 / сборка 17763) или более новая.
* Среда разработки: Visual Studio 2022 с установленной нагрузкой для разработки под Windows (UWP/WinUI) или .NET 10.0 SDK.

### Сборка и запуск

Сборка проекта:
```bash
dotnet build
```

Запуск приложения:
```bash
dotnet run
```

## Развертывание

Публикация автономного релиза (Single-File / Self-Contained):
```bash
dotnet publish -c Release -r win-x64 --self-contained true
```
Результат будет сохранен в папке `publish/` и включает в себя:
* `Screentation.exe` (автономный исполняемый файл ~300 МБ, содержащий рантайм .NET и библиотеки WinUI 3).
* Папку `Assets/` (графические ресурсы приложения).

### Предупреждение Windows Defender SmartScreen
Поскольку исполняемый файл приложения не подписан цифровым сертификатом разработчика, при первом запуске Windows Defender SmartScreen может заблокировать его.
Чтобы запустить приложение:
1. Нажмите ссылку **Подробнее**.
2. Нажмите кнопку **Выполнить в любом случае**.
